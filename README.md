# HIIT / Tabata Timer

ミニマリストデザインのHIIT（高強度インターバルトレーニング）/ タバタ式トレーニング用タイマーアプリです。

## 機能
- **ミニマリスト設定**: トレーニング時間、休憩時間、セット数を簡単に設定。
- **視覚フィードバック**:
  - 作業中 (Work): 赤色の背景
  - 休憩中 (Rest): 青色の背景
  - 完了 (Complete): 緑色の背景
  - 巨大なタイマー表示と進捗バー
- **聴覚フィードバック**: 3, 2, 1のカウントダウン音と、フェーズ切り替え時のビープ音。

## 使い方

### 1. ローカルで起動する (PCのみ)
```bash
npm install
npm run dev
```

### 2. スマホからアクセスする (同じWi-Fi内)
PCでアプリを起動した状態で、コンソール（ターミナル）に表示される `Network` のURLを確認してください。

例:
```
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.1.5:5173/  <-- このURLをスマホで開く
```

**注意**: PCとスマホが同じWi-Fiネットワークに接続されている必要があります。

### 3. Web上に公開する (Vercel)
誰でもアクセスできるようにするには、Vercelなどのホスティングサービスを使うのが最も簡単です。

1. GitHubにこのリポジトリをプッシュする。
2. [Vercel](https://vercel.com/) にログインし、"Add New Project" を選択。
3. GitHubリポジトリを選択して "Deploy" をクリック。
4. 数秒でデプロイが完了し、公開URLが発行されます。
